---
title: "Change password"
description: "Update a user's password by verifying their current password and Gmail address."
---

`PATCH /users/change-password`

Allows a user to change their password by providing their Gmail address, current password, and the new password they wish to set. All parameters are passed as query strings.

<Note>
  No authentication required. This is a self-service endpoint identified by Gmail address.
</Note>

## Parameters

<ParamField query="gmail" type="string" required>
  The Gmail address of the account whose password should be updated.
</ParamField>

<ParamField query="old_password" type="string" required>
  The user's current password. Must match the stored password to proceed.
</ParamField>

<ParamField query="new_password" type="string" required>
  The replacement password. Must meet all of the following requirements:

  - Minimum 8 characters
  - At least one uppercase letter (A–Z)
  - At least one lowercase letter (a–z)
  - At least one digit (0–9)
  - Must differ from the current password
</ParamField>

## Example

<CodeGroup>

```bash cURL
curl --request PATCH \
  --url 'https://api.example.com/users/change-password?gmail=john%40example.com&old_password=Secret123&new_password=NewPass456'
```

</CodeGroup>

## Response

<ResponseField name="status" type="string" required>
  Always `"ok"` on success.
</ResponseField>

<ResponseField name="message" type="string" required>
  Human-readable confirmation message.
</ResponseField>

```json 200 Success
{
  "status": "ok",
  "message": "Contraseña actualizada correctamente"
}
```

## Error codes

| Code | Cause |
| ---- | ----- |
| `400` | One or more required fields (`gmail`, `old_password`, `new_password`) are missing. |
| `404` | No account found with the provided Gmail address. |
| `401` | The current password (`old_password`) is incorrect. |
| `401` | The new password does not meet the format requirements (min 8 chars, 1 uppercase, 1 lowercase, 1 digit). |
| `401` | The new password is identical to the current password. |
| `500` | The database update failed — the password was not changed. |
